# Revised appendix consistency boundary

The authoritative mathematical sources remain external to Git. Their verified
filenames, hashes, IDs, and labels are recorded in repository metadata without
copying appendix source text.

Canonical execution distinguishes exact laws, numerical approximations,
clipped diagnostics, singular/blocked states, proxy-only backends, and
`numerical_candidate` closure. See:

- `docs/tne_appendix_repository_consistency_report.md`;
- `docs/tne_decision_ledger.md`;
- `docs/tne_source_law_registry.md`;
- `docs/tne_open_implementation_gaps.md`.
