# Theorem-complex traceability

The complete traceability table is
`docs/data/theorem_complex_implementation_matrix.csv`. It contains all 351
verified appendix IDs and labels plus implementation, test, simulation,
visualization, artifact, and decision status.

Use `docs/tne_theorem_complex_implementation_status.md` for the human-readable
coverage report, `docs/tne_source_law_registry.md` for source authority, and
`docs/data/artifact_provenance_manifest.json` for implemented-complex artifacts.

Status is fail-closed: a legacy/proxy file cannot upgrade a theorem row, and a
finite residual cannot by itself establish closure, attainment, or proof.
