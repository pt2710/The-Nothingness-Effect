## **2. Theoretical Foundation**
    
### **Definition of the Flowpoint Function**
    
The Flowpoint function \( fp \) is defined as an oscillatory function that alternates between positive and negative states:
    
\[
fp(f) = (f \neq -f)^{(f = -f)}
\]
    
**Oscillation Property:**
    
\[
f_{n+1} = -f_n
\]
    
Each call to \( fp(f) \) alternates the sign of \( f \).
    
### **Modeling Basic Mathematical Operations Using \( fp \)**
    
#### **1. Addition**
    
\[
fp(a + b) = fp(a) + fp(b)
\]
    
#### **2. Subtraction**
    
\[
fp(a - b) = fp(a) - fp(b)
\]
    
#### **3. Multiplication**
    
\[
fp(a \times b) = fp(a) \times fp(b)
\]
    
#### **4. Division**
    
\[
fp\left( \frac{a}{b} \right) = \frac{fp(a)}{fp(b)}
\]
    
#### **5. Exponentiation**
    
\[
fp(a^n) = \left( fp(a) \right)^n
\]
    
#### **6. Square Root**
    
\[
fp\left( \sqrt{a} \right) = \sqrt{fp(a)}
\]

    - Instantiate the FlowpointMath class and use its methods to compute values.
    Example:

        result = fp_math.fp_addition(3, 5)

# Import the fp function

from flowpoint import fp

class FlowpointMath:
    def __init__(self):
        """
        Initializes the FlowpointMath class by setting up a dictionary to manage generators for each unique input.
        """
        self.generators = {}

    def fp_value(self, f):
        """
        Retrieves the next value in the oscillation sequence for a given input f.

        Parameters:
            - f (float): Input value.

        Returns:
            - float: Next oscillatory value of f.
        """
        if f not in self.generators:
            self.generators[f] = fp(f)
        return next(self.generators[f])

    def fp_addition(self, a, b):
        """
        Models addition using the Flowpoint function.

        Parameters:
            - a (float): First operand.
            - b (float): Second operand.

        Returns:
            - float: Result of fp(a + b) = fp(a) + fp(b).
        """
        fp_a = self.fp_value(a)
        fp_b = self.fp_value(b)
        return fp_a + fp_b

    def fp_subtraction(self, a, b):
        """
        Models subtraction using the Flowpoint function.

        Parameters:
            - a (float): Minuend.
            - b (float): Subtrahend.

        Returns:
            - float: Result of fp(a - b) = fp(a) - fp(b).
        """
        fp_a = self.fp_value(a)
        fp_b = self.fp_value(b)
        return fp_a - fp_b

    def fp_multiplication(self, a, b):
        """
        Models multiplication using the Flowpoint function.

        Parameters:
            - a (float): First operand.
            - b (float): Second operand.

        Returns:
            - float: Result of fp(a * b) = fp(a) * fp(b).
        """
        fp_a = self.fp_value(a)
        fp_b = self.fp_value(b)
        return fp_a * fp_b

    def fp_division(self, a, b):
        """
        Models division using the Flowpoint function.

        Parameters:
            - a (float): Numerator.
            - b (float): Denominator.

        Returns:
            - float: Result of fp(a / b) = fp(a) / fp(b).

        Raises:
            - ZeroDivisionError: If fp(b) results in zero.
        """
        fp_a = self.fp_value(a)
        fp_b = self.fp_value(b)
        if fp_b == 0:
            raise ZeroDivisionError("Division by zero encountered in fp_division.")
        return fp_a / fp_b

    def fp_exponentiation(self, a, n):
        """
        Models exponentiation using the Flowpoint function.

        Parameters:
            - a (float): Base.
            - n (float): Exponent.

        Returns:
            - float: Result of fp(a^n) = (fp(a))^n.
        """
        fp_a = self.fp_value(a)
        return fp_a ** n

    def fp_square_root(self, a):
        """
        Models square root using the Flowpoint function.

        Parameters:
            - a (float): Input value.

        Returns:
            - float: Result of fp(sqrt(a)) = sqrt(fp(a)).

        Raises:
            - ValueError: If fp(a) is negative.
        """
        fp_a = self.fp_value(a)
        if fp_a < 0:
            raise ValueError("Cannot compute square root of negative number in fp_square_root.")
        return math.sqrt(fp_a)
